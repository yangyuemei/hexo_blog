hexo博客部署到coding

  1. 安装node.js 环境，用命令行工具node -v npm -v检测是否安装成功

  2. 安装git 环境，用命令行工具git --version 检测是否安装成功

  3. git全局配置：

    • git config --global user.name "your name"
    • git config –global user.name “your email;” `

  1. 注册coding账号并登陆,新建项目比如:myblog

    • 切换到pages->部署来源选master分支

  2. 配置 sshkey

    • 使用命令工具 ssh-key -t rsa -C "your@com"
    • 一般在c盘打开.ssh->id_rsa.pub 复制里面全部的内容到coding找到ssh粘贴添加即可
    • 测试ssh是否添加成功，ssh -T git@git.coding.net,出现hi ，‘你的姓名就是成功

  3. 安装hexo

    • npm install -g hexo
    • npm install hexo --save
    • 看看hexo是否安装成功 hexo -v

  4. 随便在本地的一个地方新建一个文件夹hexo，进入到它的根目录，右键 Git Bash Here 进入

  5. 初始化hexo ：hexo init

  6. npm 自动安装需要的组件：npm install

  7. 生成 hexo g

  8. 启动 hexo d

  9. 若端口冲突。则hexo server -p 5000

  10. 到此为止，hexo在本地的配置已经完成了

  • root：/你在coding建的仓库名/
